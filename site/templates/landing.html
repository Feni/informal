Powered by WebAssembly

WebAssembly is a new binary format for code that enables high performance applications in browsers and on servers. Arevel code compiles to WebAssembly on the server for fast response times and secure, sandboxed code execution. In the future, you will also be able to interop with other languages that compile down to WebAssembly.

Interactive, Real-time Development

Arevel combines the power & flexibility of a modern programming language with the simplicity of a spreadsheet. Explore and change your application backend with an editor that connects you seamlessly to your code, data and infrastructure.

Serverless Scaling

Deploy changes instantly and scale up to meet the demands of your app without needing to manage any server infrastructure. Arevel's compiled format allows automatic horizontal scaling with low overhead.
