Arevel is a visual language and serverless platform to build, launch and scale dynamic web applications.

Interactive, Real-time Development

Explore and modify your application backend more effectively with an editor that connects you to directly to your code and data. Arevel combines the power and flexibility of a high level programming language with the visual ease-of-use of a spreadsheet.

Powered by WebAssembly

WebAssembly is a new binary code format that enables high performance applications in browsers and on servers. Arevel code compiles down to a portable WebAssembly format for fast response times and secure, sandboxed code execution. In the future, Arevel will also be able to introp with any language that compiles down to WebAssembly.

Serverless Scaling

Deploy changes instantly and scale up to meet the demands of your app without needing to manage any server infrastructure. Arevel's compiled format allows rapid horizontal scaling without being pinned to individual servers when idle.
