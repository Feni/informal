A new medium of thought

“A computer language is not just a way of getting a computer to perform operations but rather that it is a novel formal medium for expressing ideas about methodology. Thus, programs must be written for people to read, and only incidentally for machines to execute.”

Structure and Interpretation of Computer Programs

Like spoken languages, a programming language is a medium of thought and expression. Your programming language shapes the way you approach problems. Traditional languages focus on parsing, compiling and executing code efficiently on a computer. Arevel focuses on your freedom to express ideas, solve problems and think creatively.

Coding evolved

Friendly and easy to use

The Arevel language is built on a few simple ideas that are easy to learn for both new programmers and expert developers alike. Arevel editor is fully browser based, so you can get started quickly and access your coding environment seamlessly from all of your devices.

Immediate connection to runtime

Arevel gives you a direct connection to the system you're building, allowing you to see and interact with every part of your program. This direct, observable state makes programs easier to understand and change than abstract constructs.

Code ❤ Data

Arevel melds code and data together into one, integrated system. You can work with your data with a combination of code and manual interactions.
